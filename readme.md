# 🧹 LeetSniff

A Python script that uses Playwright to scrape data from LeetCode Contests after logging in with a saved session (`auth.json`).

---

## ⚙️ Requirements

- Python 3.7+
- `auth.json` from your login session
- `playwright` package (and installed browsers)
